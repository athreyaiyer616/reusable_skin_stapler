# Reusable Skin Stapler for Low and Middle-Income Countries

This is the repository containing all of the parts required to assemble a reusable skin stapler. This project was done through a collaboration between Virginia Tech and the Malawi University of Science and Technology, however it is designed for anyone to pick this up and assemble it. 

Currently all of the parts are as SLDPRTs, so a SOLIDWORKS subscription is required. This repository will be updated with STEP and STL files, so the subscription isn't required.

This repository is currently a preliminary DRAFT. Final files will be updated in a future commit.

## Assembly

To assemble the stapler, as well as to find more information, please reference this instruction manual: [Instruction Manual](https://github.com/athreyaiyer616/reusable_skin_stapler/blob/main/Preliminary%20Draft-Instruction%20Manual%20-%20Reusable%20Skin%20Stapler.pdf)
